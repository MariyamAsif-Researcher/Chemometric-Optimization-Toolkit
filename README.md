# Chemometric Optimization Toolkit

A comprehensive Python-based toolkit for chemometric analysis, statistical modelling, Response Surface Methodology (RSM), Artificial Neural Networks (ANN), and optimization of microwave-assisted extraction processes.

---

## Project Overview

This repository demonstrates a complete computational workflow for analysing microwave-assisted extraction experiments using modern chemometric and machine learning techniques. The toolkit was developed as a research-oriented portfolio project inspired by optimization studies involving Natural Deep Eutectic Solvents (NaDES) and antioxidant extraction.

The project integrates statistical analysis, visualization, regression modelling, response surface methodology, artificial neural networks, and desirability-based optimization into a reproducible Python workflow.
